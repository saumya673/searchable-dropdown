# Searchable Country Dropdown (React + Vite)

A simple **React + Vite** project that implements a **searchable dropdown component** for countries.

The country list is served from a local mock API using **json-server**, and filtering is done on the **client side** as the user types.

---

## Features

- Searchable country dropdown
- Client-side filtering (prefix-based search)
- Country data fetched from a mock API
- Local mock server using `json-server`
- Built with React + Vite + TypeScript

---

## Tech Stack

- **React**
- **Vite**
- **TypeScript**
- **json-server**
- **pnpm**

---
